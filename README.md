## 依赖环境

1. PHP7.0 版本及以上（低版本和7.4版本未做兼容处理！）


# 安装
## 配置源库

```text
"repositories": {
        "packagist": {
            "type": "composer",
            "url": "https://mirrors.aliyun.com/composer/"
        }
    }
```

## 安装

```text
composer require liguangchun/bt -vvv
```

## 更新

```text
composer update liguangchun/bt -vvv
```

## 删除

```text
composer remove liguangchun/bt -vvv
```
